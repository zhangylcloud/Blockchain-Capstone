# Udacity Blockchain Capstone

The capstone will build upon the knowledge you have gained in the course in order to build a decentralized housing product. 

## Test steps
1. install truffle and ganache-cli 
2. ```git clone https://github.com/zhangylcloud/Blockchain-Capstone.git```
3. ```cd eth-contracts``` 
4. ```npm install```
5. ```truffle test```, we should see truffle tests pass for ERC721Mintable, Verifier and SolnSquareVerifier contract.
